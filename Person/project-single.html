The Bubble Machine

Sketching and Brainstorming Process

The idea is to create a simple interactive toy that would bring delight to the user. The machine would act aside a mini gameboy that could play a simple yet difficult brick breaking game using an LED matrix panel and a potentiometer. As the game progresses, the LEDs on the top light up with every hit of the ball as shown in the sketch below. If you manage to hit all the lights within a minute you win the game. A blow of bubbles and flashing LEDs would announce your victory! When the machine is in zen mode - another mode for the toy, it would gently blow bubbles every five seconds and play relaxing wind chimes using a mini speaker.

The following are the key features the project could address:
- tell time
- play some 8bit minimal game (avoiding obstacles)
- play music as it blows bubbles (piano or zen wind chimes)
