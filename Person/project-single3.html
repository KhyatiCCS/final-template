The Bubble Machine

Working Prototype Progress

I started working on the physical working prototype but a few of the components were missing mainly - Raspberry pie hat, soundboard, cables for a 16x32 RGB matrix and some buttons. I also learnt Python for a week to be able to program the raspberry pi but unfortunately had to abandon the idea and work with Arduino MKR1000 to work the machine. The current model has the bubble machine functioning as required. There were a few issues while working the motors:
- The DC motor libraries were corrupted and thus I couldnt program it to pulse on and off.
- The code for switching it on and off vis buttons also would not switch off the motor.
- This led the fan to be switched on all the while it was connected.

The next steps for the project would be:
1. Work the DC motor to pulse on and off
2. Wire and code the game using a library
3. Connect the sound board and load the music file and add it to the code
4. Edit the code to make the buttons switch the bubble machine and led game on and off
5. Test and iterate
6. Construct a physical structure to the machine.
